# assets/characters

Os personagens da marca da Ju&Vi Collab **não** acompanham o clone público
(material de marca proprietário). Para rodar com os visuais originais, adicione
aqui os PNGs referenciados pelas telas (ex.: `char-grupo.png`, `char-cloud.png`,
`char-calma-full.png`, `icon-calma.png`, etc.).

Sem esses arquivos o app ainda compila e roda em **modo de demonstração**;
apenas os espaços de imagem ficam vazios.
